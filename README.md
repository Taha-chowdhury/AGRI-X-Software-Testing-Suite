## 📌 Project Overview

This repository contains a **collection of data science projects** focusing on:
- 📱 Behavioral analysis (Social Media vs Student Performance)
- 🚢 Survival prediction insights (Titanic dataset)
- 📚 Web scraping and dataset building (Books dataset using R)

The goal is to demonstrate **end-to-end data science workflow** including:
- Data collection
- Cleaning & preprocessing
- Statistical analysis
- Visualization
- Insight generation

---

## 📊 Project Modules

### 📱 1. Social Media Academic Impact Analysis (Python)
- Hypothesis testing (T-test)
- Effect size (Cohen’s d)
- Correlation analysis
- Advanced visualization (Seaborn, Plotly)

### 🚢 2. Titanic Survival Analysis (R)
- Missing value handling
- Outlier detection
- Feature engineering
- Class imbalance handling
- Survival pattern analysis

### 📚 3. Books Web Scraping Pipeline (R)
- Multi-page scraping using rvest
- Feature extraction (price, rating, category)
- Dataset creation (1000+ records)
- Data structuring & cleaning
